Estruturas de controle:

Condicionais:

if (condição){
 //verdadeira
}


if (condição){
 //verdadeira
} else{
 //falsa
}

Obs1: Na estrutura if else uma das condições será escrita
Obs2: Nunca as duas condições serão escritas simultaneamente

if (condição1){
 //condição1 verdadeira
} else if (condição2) {
 //condição2 verdadeira
} else {
 //caso nenhuma condição seja verdadeira
}

Obs1: Esta é uma única estrutura
Obs2: Ao encontrar uma condição verdadeira a estrutura é encerrada
Obs3: O else final não é obrigatório
